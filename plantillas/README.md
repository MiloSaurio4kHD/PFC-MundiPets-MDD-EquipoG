# plantillas/

Plantillas Apache Velocity (`.vm`) empleadas por el Instant Generator de Visual Paradigm para generar el código C#, junto con la captura de la configuración utilizada.

| Plantilla | Responsabilidad |
|---|---|
| `class.vm` | Estructura general de la clase |
| `attribute.vm` | Generación de atributos y propiedades |
| `operation.vm` | Generación de métodos con su firma |
| `association.vm` | Traducción de asociaciones a campos |
| `parameter.vm` | Parámetros de las operaciones |
| `enum.vm` | Declaración de enumeraciones |
| `auto-implemented-property.vm` | Propiedades con `{ get; set; }` |

`ConfiguracionInstantGeneratorMundiPets.png` muestra los parámetros de generación aplicados.
