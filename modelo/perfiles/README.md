# modelo/perfiles/

Perfil UML del proyecto, que demuestra el mecanismo de extensión de UML mediante estereotipos y valores etiquetados (tagged values).

| Archivo | Descripción |
|---|---|
| `MundiPets-DiagramaPerfil.vpp` | Proyecto de Visual Paradigm con el diagrama de perfil |
| `MundiPets.DiagramaPerfil.png` | Exportación del diagrama de perfil |

El perfil define estereotipos que extienden las metaclases `Class` y `System`. El estereotipo `Entity` se aplica sobre las clases del dominio en el diagrama de clases.
