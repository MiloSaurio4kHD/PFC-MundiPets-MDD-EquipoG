# modelo/

Modelo UML de origen en formato nativo de Visual Paradigm.

| Archivo | Descripción |
|---|---|
| `MundiPets-DiagramaClases.vpp` | Proyecto de Visual Paradigm con el diagrama de clases del subsistema |
