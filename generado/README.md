# generado/

Código C# producido por el Instant Generator de Visual Paradigm a partir del diagrama de clases.

- **Archivos `.cs` sueltos:** salida directa del generador (8 clases + 2 enumeraciones).
- **`MundiPets_Evidencia_CodigoGeneradoVPP/`:** solución de Visual Studio 2022 con el código integrado, listo para compilar y ejecutar.

Nota: las clases `MascotaDTO` y `UsuarioDTO` y los ajustes de compilación (como `using System.Collections.Generic;`) se agregaron manualmente, ya que el generador produce únicamente la estructura del modelo.
